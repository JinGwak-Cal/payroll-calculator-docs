# C-00017 : 컨텍스트 쓰레시홀드 지수 (Context Threshold Index, CTI)

| 그룹 | 필드 | 내용 |
|---|---|---|
| **식별** | Concept ID | C-00017 |
| | Coordinate | M-01 |
| **기본** | 이름(한글) | 컨텍스트 쓰레시홀드 지수 |
| | 영문명 / 약어 | Context Threshold Index (CTI) |
| **정의** | 공식 정의 | 프로젝트가 Context Threshold(C-00016)에 얼마나 근접했는지를 복수의 관찰 변수를 종합해 나타내는 복합 지수 |
| | 목적 | Context Threshold를 운영 판단에 활용할 수 있도록 정량화한다 |
| | 핵심 질문 | 지금 이 프로젝트의 운영 Context는 AI 자율 판단이 가능한 수준에 얼마나 가까이 왔는가 |
| **분류** | Type | Metric |
| | Status | Candidate |
| | Stability | Experimental *(산식·변수 미확정 — HDR(C-00034)을 1차 입력값 후보로만 합의된 상태)* |
| **관계** | 상위 개념 | C-00016 (컨텍스트 쓰레시홀드) |
| | 하위 개념 | C-00034 (HDR) *[Forward Reference — 미작성, ID 예약]* |
| | 관련 개념 | C-00018 (클로드 관찰 사례 — CTI의 최초 경험적 근거) |
| | Before(전 단계) | C-00018 (관찰) → C-00016 (가설) → **CTI(측정)** 순서 |
| | After(후 단계) | C-00016 (판단 — CTI 값이 가설 검증에 피드백됨) |
| **비교** | 기존 유사 개념 | KPI (Key Performance Indicator), NPS (Net Promoter Score), Maturity Score |
| | 차이점 | 일반 KPI는 달성률을 측정하지만, 본 지수는 "AI 자율성 발생"이라는 임계적 상태 전이에 얼마나 근접했는지를 측정한다 — 달성/미달성의 이분법이 아니라 연속적 근접도를 다룬다 |
| | 용어 유형 | 기존용어조합+신개념 |
| | Semantic Boundary | 업계에서 Index는 보통 여러 지표를 종합한 복합 지수(예: 소비자물가지수)를 의미하며, Score는 단일 측정 결과를 의미한다. CTI는 복수 변수(HDR 등)를 종합할 예정이므로 Index가 맞다 — Score로 부르면 단일 측정으로 오해될 수 있다 |
| | Official Name 후보 | 없음 — Index/Score 구분 명확히 하여 Index로 확정 |
| **근거** | Origin Thread | Research Thread #001 |
| | Discovery Context | Context Threshold(C-00016) 가설이 정성적 관찰에만 머무르는 것을 넘어 측정 가능한 지표로 발전시키려는 과정에서, HDR을 1차 측정 변수 후보로 삼아 Index 개념으로 정의됨 |
