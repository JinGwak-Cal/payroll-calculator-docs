# Event Card Template
복사해서 바로 작성. 설명·슈궍기준·Lifecycle·운영철학은 여기 넣지 않음(Handbook 참고).

**제목 규칙:** Event Label은 "무엇을 논의했는가"가 아니라 "무엇이
확정되었는가"를 기준으로 작성한다(예: "교정표 수정" ❌ →
"Default Correction Table v6 확정" ⭕).

**배치 헤더 필수:** 이 카드를 `context-history.md`에 추가하기 전,
같은 세션(쓰레드)에서 처음 추가하는 것이라면 먼저 아래 배치 헤더부터
만든다. 이미 오늘 배치가 있으면 그 아래에 카드만 이어 붙인다.

```
## 배치: YYYY-MM-DD
**Source Threads**
- Claude: (쓰레드 제목)
- GPT: (쓰레드 제목, 있는 경우만)
```
