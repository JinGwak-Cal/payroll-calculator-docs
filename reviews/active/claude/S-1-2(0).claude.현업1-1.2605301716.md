## 문구 변경 확정안

### 1단 — 공통 상단 안내 (신규 추가, 주휴 섹션 위 통짜 한 칼럼)
```
첫 주·마지막 주 주휴 반영은 이용자 선택으로 결정됩니다.
주휴수당 판단: 주간 개근 + 주 15시간 이상 근무
근무시간은 입력한 수치를 기준으로 자동 반영됩니다.
```

### 2단 — 카드 섹션
허락된 변경: 토글 아래 (개근·N시간) 문구 삭제
그 외 일체 무변경 (UI/테두리/tailwind/구조 전부)

### 3단 — 하단 결과 문구
위치 변경 (내용·형식 그대로, 위치만):
```
현재: ◎ 주휴 발생 가능
      개근·주간 20시간     ← 아랫줄
변경: ◎ 주휴 발생 가능 (개근·주간 N시간)  ← 같은 줄 괄호
```

삭제:
```
※ 주휴수당 발생 여부는 사업장 기준에 따릅니다
※ 모든 주는 월~일 기준으로 판단됩니다 (첫주·마지막주 포함)
```

미발생 문구:
```
◎ 주휴 미발생 (15시간 미달)
```
